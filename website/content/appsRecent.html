ReAR View (Recent App Releases)

This tool allows you to view recent releases of core apps on GitHub.

The period selection defaults to the last month. Click "Show releases" to display the changes for the period.
The default view shows all commits linked to a Jira issue. Toggle "Show all commits" if you want to see those without a Jira ticket too.
Alternatively you can view the GitHub release text for all of the releases in the period, by toggling that option.
