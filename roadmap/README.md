# DHIS2 Roadmap

This folder holds a small, self-contained web page that presents the **DHIS2 feature
roadmap** drawn from the Jira `ROADMAP` project. The page is plain HTML + JavaScript with no
build step, served from GitHub Pages and embedded as an iframe on the DHIS2 website.

The page reads its content at runtime from [jira_roadmap.json](jira_roadmap.json), so the
page itself rarely changes — refreshing the roadmap means regenerating that file.

## Page

| Page | Embedded at | Reads | Script / Style | What it shows |
|------|-------------|-------|----------------|---------------|
| [product-view.html](product-view.html) | _https://dhis2.org/roadmap/_ | [jira_roadmap.json](jira_roadmap.json) | [resources/product-view.js](resources/product-view.js), [resources/product-view.css](resources/product-view.css) + [resources/styles.css](resources/styles.css) | Roadmap organised into product-area tabs (Platform, Configure, Collect, Analyze, Extend), filterable by release. |

## Data file

- [jira_roadmap.json](jira_roadmap.json) — Snapshot of the Jira `ROADMAP` project: a `metadata`
  block (project, timestamp, issue count) plus an `issues` array (key, summary, status,
  issue type, target release, product areas, etc.). Generated by the roadmap fetcher in
  [tools/roadmap/](../tools/roadmap/) (`process_roadmap.py`).

## Styles

- [resources/styles.css](resources/styles.css) — Base roadmap styling.
- [resources/product-view.css](resources/product-view.css) — Additional styling for the
  product-view tabs layout.
