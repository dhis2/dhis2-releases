# Patch 42.5.1 Release Note

This is a HOTFIX patch release to address the following security advisories:

- **[GHSA-pwmg-mvjw-4m23](https://github.com/dhis2/dhis2-core/security/advisories/GHSA-pwmg-mvjw-4m23): SQL injection vulnerability**

- **[GHSA-6785-hj47-c27h](https://github.com/dhis2/dhis2-core/security/advisories/GHSA-6785-hj47-c27h): Reflected cross-site scripting (XSS) vulnerability**

- **[GHSA-3fr2-wvqx-cmr5](https://github.com/dhis2/dhis2-core/security/advisories/GHSA-3fr2-wvqx-cmr5): Unsafe Java deserialization vulnerability**

It also bundles the following stability fix:

- **[DHIS2-21604](https://dhis2.atlassian.net/browse/DHIS2-21604): Generating a Personal Access Token causes DHIS2 42.5 service corruption**

See [Patch 42.5.0](ReleaseNote-2.42.5.md) for the full list of fixes included in the parent patch release.
